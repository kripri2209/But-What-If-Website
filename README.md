# Read me - Devils-Advocate-Website

24 February 2026

I understood the problem as designing a tool where a user can enter a decision, compare multiple options, define what factors matter to them, and receive a structured evaluation with clear reasoning. I assumed that users may not always have complete information and that their decisions could be influenced by bias or emotion, so the system should highlight risks and challenge assumptions rather than operating as a black box.

And so I came up with 'DEVIL'S ADVOCATE": my goal was to create a system that helps users think more critically about their decisions rather than simply giving them an answer. It challenges the user based on the input and actually cross-questions them, instead of blindly accepting the user's choice.
